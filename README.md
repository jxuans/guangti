## 羽毛球场地自动化预约实现思路

### 需求分析

1.脚本需定时运行，在计划打球两天前的早上八点运行，单次预约失败则重试，一定失败次数后结束。



2.成功与否，脚本结束时都发送邮件通知。



3.脚本运行顺序：统一身份认证平台登录 - 羽毛球场地预约 - 邮件推送

### 项目组成

- 统一身份认证登录程序：实现登录到体育场馆预约系统
- 场地预约程序：根据提前设定的日期，按一定顺序选择场地，并向服务器发送请求
- 邮件推送程序：根据预约结果，发送邮件反馈。

### 开发时间

2-3周
